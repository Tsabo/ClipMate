# Bitmap Tab

The Bitmap Tab displays bitmap image clips. These include screenshots, photos, and any pixel-based image data captured to the clipboard.

## Viewing Options

The Bitmap Tab offers several viewing modes:

| Mode | Description |
|------|-------------|
| **Full Size** | Shows the image at 100% in a scrolling window |
| **Fit** | Scales the image to fit the available space while maintaining aspect ratio |
| **Stretch** | Stretches the image to fill all available space |

## Image Information

The tab displays the X,Y dimensions (width × height in pixels) of the image.

## Tips

:::tip Larger viewing area
Undock the Preview/Edit window for a larger viewing area. You can even maximize an undocked viewer.

The quickest method:
1. Press **F2** to open a new Preview window
2. Maximize if you need to see more
3. Close the window when done (don't minimize—close it)
:::
