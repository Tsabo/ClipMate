# Text Tab

The Text Tab displays and edits TEXT items. It features a toolbar full of powerful editing functions that rival stand-alone editors.

## Editing Features

You can edit text clips using:
- Direct typing
- Drag and drop
- Cut/Copy/Paste

## Editor Toolbar Functions

The editor toolbar provides quick access to many text manipulation features:

| Function | Description |
|----------|-------------|
| **Tack** | Pins the current clip in the editor |
| **Word Wrap** | Toggles word wrap on/off |
| **Non-Printing** | Shows spaces, tabs, line endings |
| **New Clip** | Creates a new empty clip |
| **Cut/Copy/Paste** | Standard clipboard operations |
| **Shift Left/Right** | Adjusts indentation |
| **Remove Line Breaks** | Joins lines together |
| **Change Case** | Upper, lower, title case |
| **Text Cleanup** | Various cleanup operations |
| **Trim** | Removes leading/trailing whitespace |
| **Find/Replace** | Search and replace text |
| **Undo** | Reverses last edit |
| **Spell Check** | Checks spelling |
| **Highlight URLs** | Makes URLs clickable |
| **Highlight HTML** | Syntax highlights HTML tags |
| **Encrypt/Decrypt** | Secures sensitive data |
| **Base64** | Encode/decode Base64 |

## Rectangular Selections

The text editor supports rectangular (column) selections, useful for working with tabular data or code. Hold **Alt** while dragging to make a rectangular selection.

## Tips

:::tip Toolbar visibility
Right-click within the editor to toggle the toolbar on or off. This is especially useful when the editor is docked in ClipMate Classic where space is limited.
:::

:::tip Customize toolbar
Right-click on the toolbar and select "Customize" to choose between large and small button sets.
:::
