# Show ClipMate

This button is only available on the ClipBar and causes either ClipMate Classic or ClipMate Explorer to be displayed. It brings up whichever window was viewed last during the current session.

## Usage

Click the Show ClipMate button on the ClipBar to:
- Display ClipMate Classic if it was last used
- Display ClipMate Explorer if it was last used

## Alternative Methods

You can also show ClipMate using:

| Method | Action |
|--------|--------|
| System Tray | Click the ClipMate icon in the system tray |
| Hotkey | Press **Ctrl+Alt+C** (configurable) |
| ClipBar Drop-down | Click the drop-down arrow on the ClipBar |
