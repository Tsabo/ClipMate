This Explorer-only button toggles the ClipList between Detail View and Thumbnail View.

## Views

### Detail View (Default)
Shows clips in a multi-column list with sortable columns:
- Title
- Date/Time
- Creator (source application)
- Source URL
- Available formats
- And more

### Thumbnail View
Displays bitmap and text clips as graphic thumbnails, providing a visual preview of clip contents. This is especially useful when working with:
- Clip art collections
- Screenshots
- Image libraries

## Usage

Click the **View Mode** button (at the far right of the Explorer toolbar) to toggle between views.
