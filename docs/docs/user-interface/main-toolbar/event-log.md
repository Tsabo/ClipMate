The Event Log button opens the Event Log window, which displays messages, warnings, and errors generated by ClipMate during operation.

## What It Shows

The Event Log displays:
- **Informational messages** - Normal operations and status updates
- **Warnings** - Potential issues that don't prevent operation
- **Errors** - Problems that may affect functionality

Each entry includes:
- Timestamp
- Severity level (Info, Warning, Error)
- Message description

## When to Use

The Event Log is useful for:
- Troubleshooting problems
- Monitoring ClipMate's activity
- Diagnosing capture or paste issues
- Identifying configuration problems

## Event Log Features

- **Clear** - Remove all entries
- **Copy** - Copy selected entries to clipboard
- **Export** - Save log to file for support

## Accessing Event Log

You can open the Event Log in multiple ways:
1. Click the Event Log toolbar button
2. From the menu: **Tools** > **Event Log**
3. Keyboard shortcut (if configured)
