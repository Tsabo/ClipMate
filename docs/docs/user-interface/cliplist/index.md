# The ClipList

The ClipList is the central pane where clips are displayed in ClipMate. It shows the clips contained in the currently selected collection or search results.

## View Modes

The ClipList supports multiple view modes to suit your workflow:

### Detail View
Shows clips in a list format with columns for:
- Icon (clip type)
- Title/content preview
- Date captured
- Source application
- Source URL (if available)
- Size

Columns can be resized, reordered, and shown/hidden.

### Thumbnail View
Shows clips as visual previews in a grid. Particularly useful for:
- Image clips - see the actual image
- Text clips - see a preview of the content
- HTML clips - see rendered preview

Thumbnail size can be adjusted.

## Selecting Clips

- **Single click** - Select a clip
- **Ctrl+Click** - Add clip to selection
- **Shift+Click** - Select range of clips
- **Ctrl+A** - Select all clips

## Sorting

Clips can be sorted by any column by clicking the column header. Click again to reverse the sort order. See [Sorting Clips](sorting-clips.md) for details.

## Context Menu

Right-click on clips to access common operations:
- Paste (various methods)
- Edit
- Delete
- Move/Copy to Collection
- Properties

## Item Type Icons

Each clip displays an icon indicating its type:
- Text
- Rich Text (RTF)
- HTML
- Images (Bitmap, Picture)
- Files
- Binary data

See [Item Type Icons](item-type-icons.md) for the complete icon reference.
