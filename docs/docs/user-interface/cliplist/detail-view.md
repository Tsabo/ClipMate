# Detail View

Detail View is the default view mode for the ClipList. It presents clips as a multi-column display with sortable columns.

## Switching Views

Click the View Mode button at the far right of the ClipMate Explorer toolbar to switch between Detail View and Thumbnail View.

## Available Columns

Detail View shows multiple columns of information:

| Column | Description |
|--------|-------------|
| **Icon** | Shows the clip type ([Item Type Icons](item-type-icons.md)) |
| **Title** | The clip title or content preview |
| **Date/Time** | When the clip was captured |
| **Source** | The application the data was copied from |
| **Source URL** | The web page URL (if applicable) |
| **Size** | Size of the clip data |
| **SortKey** | User-defined sort order value |
| **ID** | Unique clip identifier |

## Customizing Columns

You can customize the column display:
- **Resize** - Drag column borders to resize
- **Reorder** - Drag column headers to rearrange
- **Show/Hide** - Right-click column header to toggle columns

## Sorting

Click any column header to sort by that column. Click again to reverse the sort order. See [Sorting Clips](sorting-clips.md) for more details.
